Prawo Murphyego działa na ciebie jak na nikogo innego. Projekt w którym do tej pory pracowałeś wziął i przestał działać. Coś się konkretnie pochrzaniło, postanawiasz więc porzucić obecne rezozytorium, wyczyścić je do cna i tym razem sklonować nowe z centralnego repozytorium zdalnego. No to do dzieła.

$ git clone https://github.com/morelq/cw3.git

Wykonanie komendy clone z parametrem url do zdalnego repozytorium skutkuje stworzeniem kompletnej kopii zdalnego repozytorium.

$ git remote -v

Czy nie wyglądo to znajomo? Operacja klonowania ustawiła automatycznie zdalne repozytorium o nazwie origin.

$ git log -2
