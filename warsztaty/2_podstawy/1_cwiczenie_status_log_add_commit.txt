Przychodzisz do pracy po absorbującym weekendzie. Zanim zaczniesz pracę, chcesz dowiedzieć się jaki jest stan w twoim repozytorium projektowym. 
Zacznij od utworzenia nowego repozytorium lokalnego za pomocą polecenia

git init

Żeby sprawdzić stan, użyj funkcji

git status

Przeczytaj ze zrozumieniem output dla pustego repozytorium.

Teraz utwórz nowy pusty plik w twoim repozytorium i ponownie sprawdź status.
Przeczytaj ze zrozumieniem output. A następnie dodaj plik do indeksu. Użyj komendy

git add [nazwa_pliku]

Sprawdź status ponownie i ponownie przeczytaj go uważnie.

Zakomituj teraz nowo utworzony plik. Wpisz komendę

git commit -m "Opis tego co zostało zrobione"

Przeczytaj output z komita.
Teraz możesz też już sprawdzić log twojego repozytorium za pomocą komendy

git log

Utwórz jeszcze jeden plik, a w pierwszym zmień treść. Sprawdź ponownie status. 
Teraz dodaj zmiany do indeksu. Tym razem użyj komendy "dodaj wszystko":

git add .

Sprawdź status. 
Następnie zakomituj te zmiany, ale nie wpisuj parametru -m:

git commit

Git bash i tak poprosi cię o wpisanie i zatwierdzenie tytułu komita.
Tryb edycji w zależności od systemu
