Rozpoczynasz prace nad nowym zadaniem, ustalenia w zespole były takie że tworzycie brancza na każdą funkcjonalność. Twoim zadaniem jest napisać plik witający właśnie ciebie.

$ git clone https://github.com/morelq/cw5.git

Sklonuj repozytorium cw5 z Githuba.


$ git branch hello-[your-github-username]

Zaczynamy od stworzenia brancha branch którego nazywamy unikalnie hello-[your-github-username].

$ git status

Sprawdzamy stan

Houston, mamy problem, stworzyliśmy nowego brancha, ale git status dalej pokazuje, że jesteśmy na masterze

$ git branch

Bezparametrowy git branch pokaże nam wszystkie branche, które posiadamy lokalnie i podświetli ten na którym jesteśmy

No, ok nasz branch hello-[your-github-username] istnieje ale co dalej ?

$ git checkout hello-[your-github-username]

checkout na ratunek. Sprawdźmy rezultat ...

$ git status
$ git branch

Sprawdzamy stan

No i teraz wszystko się zgadza, obie komendy jednoznacznie pokazują, że jesteśmy na dobrym branczu.

$ git push origin hello-[your-github-username]

Pozostało już tylko udostępnić tego brancha innym a więc do dzieła
