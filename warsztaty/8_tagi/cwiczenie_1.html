Pracujesz wraz z zespołem nad projektem, który właśnie wchodzi na produkcję. Udało wam się wyrilisować projekt i oddać klientom do użytku i chcecie oznaczyć w historii gita ten historyczny moment.

Użyj repozytorium z poprzedniego ćwiczenia.

$ git checkout master
$ git pull
$ git tag production_release_[your-github-username]
$ git push --tags

Wejdź na brancz master i utwórz taga production_release_[your-github-username].

$ git log --oneline

Zobacz w historii jak nowy tag jest przestawiony.
