Okazało się, że zrobiłeś niestosowną literówkę w opisie komita i musisz ją zmienić. Użyj ponownie interaktywnego ribejza, żeby zmienić opis ostatniego komita.

Użyj repozytorium z poprzedniego ćwiczenia.

$ git rebase -i HEAD~1

Rozpocznij ribejz z uwzględnieniem jednego, ostatniego komita.

r 3ac1c09 komit #1

Wejdź w tryb edycji ( i lub Insert) i przy komicie wstaw r (reword).

Aby kontuować wciśnij Escape a następnie Shift+zz lub :wq.

Zostaniesz poproszony o edycję wskazanego komita. Wejdź w tryb edycji ( i lub Insert), zmień tytuł komita i wyjdź Shift+zz lub :wq.

$ git log master..issue-13-[your-github-username]

Teraz ponownie sprawdź log. Opis został zmieniony.

$ git push origin +issue-13-[your-github-username]

Wypuszuj tą zmianę. Tutaj również należy dodać parametr --force lub +.
